Dulces Delicias

Web Blog

In addition to teaching coding classes, I developed the educational material that was provided to students, ensuring that concepts were approachable and easy to follow. As part of the program, I designed the final project: a personal cooking blog that combined creativity with technical skills.

The blog project was first outlined through a functional prototype in Figma to define layout and structure. Later, I programmed the site using HTML and CSS, simplifying the code so students could reproduce the project step by step with the learning materials.

Problem Statement

Many students struggle with the leap from theory to practice when learning how to code. Without clear and achievable projects, it’s difficult to stay motivated and see progress. This project aimed to bridge that gap by providing an engaging, beginner-friendly website idea that students could adapt to their own interests.


Design Process

I started by prototyping the blog in Figma to establish the visual style and layout. Once the design was clear, I implemented it using HTML and CSS, focusing on clean structure and simplified code. The material was carefully prepared to ensure that students could follow along and successfully build their own version of the blog.

The project featured a simple blog design with a homepage, recipe entries, and a consistent visual style that students could customize. By keeping the code streamlined and the design accessible, the project encouraged creativity while reinforcing key programming concepts and introducing good habits in web structure.

The cooking blog served as an effective final project that allowed students to apply everything they had learned in a practical and enjoyable way. It not only reinforced their coding skills but also gave them a tangible website they could personalize and share.
